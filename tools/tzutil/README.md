# tzutil

A simple TZ convertor

## Dependencies

* ruby
* [activesupport](https://rubygems.org/gems/activesupport/versions/5.0.0)

## Installation

```sh
gem install activesupport
```

## Usage

### Convert current time

```sh
→ tzutil.rb
+------------------+---------------------------------+---------------------------+
| TZ               | RFC822                          | ISO8601                   |
+------------------+---------------------------------+---------------------------+
| UTC              | Thu, 03 Jan 2019 05:44:40 +0000 | 2019-01-03T05:44:40Z      |
| US/Pacific       | Wed, 02 Jan 2019 21:44:40 -0800 | 2019-01-02T21:44:40-08:00 |
| Australia/Sydney | Thu, 03 Jan 2019 16:44:40 +1100 | 2019-01-03T16:44:40+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Thu, 03 Jan 2019 00:44:40 -0500 | 2019-01-03T00:44:40-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 06:44:40 +0100 | 2019-01-03T06:44:40+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 14:44:40 +0900 | 2019-01-03T14:44:40+09:00 |
+------------------+---------------------------------+---------------------------+
```

### Convert a given datetime string

* ISO8601 format

```sh
→ tzutil.rb -s "2019-01-03T05:00:00Z"
+------------------+---------------------------------+---------------------------+
| TZ               | RFC822                          | ISO8601                   |
+------------------+---------------------------------+---------------------------+
| UTC              | Thu, 03 Jan 2019 05:00:00 +0000 | 2019-01-03T05:00:00Z      |
| US/Pacific       | Wed, 02 Jan 2019 21:00:00 -0800 | 2019-01-02T21:00:00-08:00 |
| Australia/Sydney | Thu, 03 Jan 2019 16:00:00 +1100 | 2019-01-03T16:00:00+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Thu, 03 Jan 2019 00:00:00 -0500 | 2019-01-03T00:00:00-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 06:00:00 +0100 | 2019-01-03T06:00:00+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 14:00:00 +0900 | 2019-01-03T14:00:00+09:00 |
+------------------+---------------------------------+---------------------------+
```

* RFC822 format

```sh
→ tzutil.rb -s "Thu, 03 Jan 2019 05:00:00 +0000"
+------------------+---------------------------------+---------------------------+
| TZ               | RFC822                          | ISO8601                   |
+------------------+---------------------------------+---------------------------+
| UTC              | Thu, 03 Jan 2019 05:00:00 +0000 | 2019-01-03T05:00:00Z      |
| US/Pacific       | Wed, 02 Jan 2019 21:00:00 -0800 | 2019-01-02T21:00:00-08:00 |
| Australia/Sydney | Thu, 03 Jan 2019 16:00:00 +1100 | 2019-01-03T16:00:00+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Thu, 03 Jan 2019 00:00:00 -0500 | 2019-01-03T00:00:00-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 06:00:00 +0100 | 2019-01-03T06:00:00+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 14:00:00 +0900 | 2019-01-03T14:00:00+09:00 |
+------------------+---------------------------------+---------------------------+
```

* Any format `DateTime.parse()` understands

```sh
→ tzutil.rb -s "2019-01-03 05:00:00 PST"
+------------------+---------------------------------+---------------------------+
| TZ               | RFC822                          | ISO8601                   |
+------------------+---------------------------------+---------------------------+
| UTC              | Thu, 03 Jan 2019 13:00:00 +0000 | 2019-01-03T13:00:00Z      |
| US/Pacific       | Thu, 03 Jan 2019 05:00:00 -0800 | 2019-01-03T05:00:00-08:00 |
| Australia/Sydney | Fri, 04 Jan 2019 00:00:00 +1100 | 2019-01-04T00:00:00+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Thu, 03 Jan 2019 08:00:00 -0500 | 2019-01-03T08:00:00-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 14:00:00 +0100 | 2019-01-03T14:00:00+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 22:00:00 +0900 | 2019-01-03T22:00:00+09:00 |
+------------------+---------------------------------+---------------------------+
```

### Convert datetime with an offset (in seconds)

* An hour from now

```sh
→ tzutil.rb -o 3600

+------------------+---------------------------------+---------------------------+
| TZ               | RFC822                          | ISO8601                   |
+------------------+---------------------------------+---------------------------+
| UTC              | Thu, 03 Jan 2019 06:53:28 +0000 | 2019-01-03T06:53:28Z      |
| US/Pacific       | Wed, 02 Jan 2019 22:53:28 -0800 | 2019-01-02T22:53:28-08:00 |
| Australia/Sydney | Thu, 03 Jan 2019 17:53:28 +1100 | 2019-01-03T17:53:28+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Thu, 03 Jan 2019 01:53:28 -0500 | 2019-01-03T01:53:28-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 07:53:28 +0100 | 2019-01-03T07:53:28+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 15:53:28 +0900 | 2019-01-03T15:53:28+09:00 |
```

```sh

| Australia/Sydney | Thu, 03 Jan 2019 15:54:36 +1100 | 2019-01-03T15:54:36+11:00 |
+------------------+---------------------------------+---------------------------+
| US/Eastern       | Wed, 02 Jan 2019 23:54:36 -0500 | 2019-01-02T23:54:36-05:00 |
| Europe/Zurich    | Thu, 03 Jan 2019 05:54:36 +0100 | 2019-01-03T05:54:36+01:00 |
| Asia/Tokyo       | Thu, 03 Jan 2019 13:54:36 +0900 | 2019-01-03T13:54:36+09:00 |
+------------------+---------------------------------+---------------------------+
```
